宠物屋 - 老师使用指南
====================

【游戏简介】
"宠物屋"是一个课堂互动工具，学生通过点击名字获得爱心来照顾虚拟宠物，激励积极参与课堂活动。

【宠物种类】
游戏中有4种宠物：橘猫、黑猫、柯基、边境牧羊犬
- 每个学生初始随机分配一只宠物
- 学生可以更换自己喜欢的宠物（需要清空爱心）

【核心玩法】
1. 点击学生名字 → 发射爱心动画 → 获得1颗爱心
2. 宠物状态循环：睡觉→苏醒→吃东西→玩耍→睡觉...
3. 升级规则：4颗爱心升到LV2，8颗爱心升到LV3获得毕业徽章

【老师操作指南】
================

一、开始使用
-----------
1. 打开网页 → 点击"进入游戏"
2. 创建班级：输入班级名称和目标
3. 添加学生：
   - 手动添加：输入学生姓名，一行一个
   - Excel导入：点击"导入数据"选择Excel文件
4. 系统自动为每个学生随机分配宠物

二、课堂互动
-----------
1. 单人互动：直接点击学生名字，学生获得1颗爱心
2. 批量互动：
   - 点击"多选互动"按钮
   - 勾选学生 → 点击"批量互动"
   - 或点击"全选" → "批量互动"
3. 小组互动：在小组管理页面点击"本组互动"

三、小组管理
------------
1. 创建小组（后台管理页或课堂界面均可）：  
   - 在后台 `admin.html` 里选择班级 → 输入小组名 → 点击“创建小组”，即可为该班添加空组。  
   - 在课堂界面点击“创建小组”按钮同样能完成同样操作。
2. 添加/调整成员：  
   - 后台：点击小组条目右侧的“删除”前，先在课堂界面使用“编辑成员”实现精细分配。  
   - 课堂界面：点击小组的“编辑成员”→ 勾选/取消学生 → 保存。保存后会自动写入本地存储，后台页面刷新即可看到最新人数。
3. 查看小组人数：  
   - 后台小组列表每行尾部都会显示“X 人”的徽章，表示当前组内学生数。  
   - 课堂界面的小组方块顶部同样会实时展示成员数量。
4. 删除小组：在后台或课堂界面都可删除；删除后该组学生会自动恢复为未分组状态。
5. 小组扣分：点击课堂界面的“本组扣分”用于课堂纪律管理（每名成员减少 1 颗爱心）；后台会在刷新后看到更新后的爱心数量排行。

四、商城奖励
-----------
1. 添加奖励：进入"商城管理" → 输入奖励名称、设置爱心门槛、库存 → "添加奖品"
2. 学生兑换：学生点击自己卡片 → "商城兑换" → 选择奖励 → 老师确认发放
3. 管理奖励：可以修改或删除奖励项目

五、学生更换宠物
---------------
学生可以更换宠物：
1. 点击学生卡片右上角的🔄按钮
2. 系统会提示"换宠物需要清除所有爱心，确定要换吗？"
3. 确认后清空该学生的所有爱心，可以选择新宠物
4. 用于激励学生重新开始积累爱心

六、数据管理
-----------
1. 数据导出：点击"导出数据"保存Excel文件（建议定期备份）
2. 数据导入：点击"导入数据"恢复之前的Excel文件
3. 切换班级：点击右上角"切换班级"选择其他班级

七、个性化设置
-------------
1. 修改标题：进入"系统设置" → "修改游戏标题" → 输入新标题
2. 调整升级门槛：进入"系统设置" → "宠物成长设置" → 设置LV2/L3需要的爱心数

八、一键清零功能
---------------
使用场景：学期结束、重新开始游戏
操作步骤：
1. 进入"系统设置"
2. 点击"本班进度归零"
3. 输入"归零"确认
4. 点击"确认归零"

清零内容包括：
- 所有学生的爱心数清零
- 宠物等级回到LV1
- 宠物状态回到睡觉
- 清除所有动物徽章
- 学生分配的小组取消
- 兑换历史清空

⚠️ 注意：此操作不可撤销，请谨慎使用！

九、快捷操作
-----------
- 撤销：点击"撤销"按钮撤销最近一次操作
- 返回主页：点击"返回主页"
- 刷新界面：按F5键

十、使用建议
-----------
1. 课堂应用：
   - 积极发言奖励爱心
   - 按时完成作业奖励爱心
   - 小组合作表现好集体奖励
   - 课堂纪律问题使用小组扣分

2. 奖励设置建议：
   - 小文具：贴纸、橡皮、铅笔
   - 特殊权利：免作业卡、优先选择权
   - 班级服务：当小助手、整理图书

3. 定期维护：
   - 每周导出数据备份
   - 根据班级情况调整升级门槛
   - 及时补充商城奖品库存

祝您教学愉快！
